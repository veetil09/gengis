An application for merging geographic data with information about biological sequences collected from the environment.

New Features

See kiwi.cs.dal.ca/GenGIS/Revision_History for all bug fixes.

Version 2.1.1 (August 7, 2013)

  • Global Biodiversity Information Facility (GBIF) Query plugin.
  • Support for visualizing multiple location sets (plugins currently only access first location set).
  • Updated welcome screen

Version 2.1.0 (April 30, 2013)

  • Reference Condition Analysis plugin.
  • One or more than one Shapefiles can be loaded and visuialized in GenGIS.
  • One or more than one shapefiles can be loaded on a digital map file.
  • User is able to update the desired symbology attributes for each shapefile.
  • Metadata for each shapefile is organized and shown in the properties dialog.
  • The minimap is updated to show both digital map file and shapefiles loaded in GenGIS.
  • Serialization works for a session containing shapefiles as well.

Version 2.0.2 (July 27, 2012)

  • Beta Diversity plugin
  • For nonlinear geographic axes, a tabular report of all permutations is now provided.
  • For nonlinear geographic axes, the optimal ordering which minimizes the total Euclidean distance of all location lines is displayed.
  • Major enhancements to the Alpha Diversity and Alpha Diversity Visualizer plugins.
  • Major enhancements to the Dissimilarity Matrix Viewer plugin.
  • Interface for exporting screen captures as PNG images.
  • GenGIS allows multiple tree files to be loaded at the same time.
  • GenGIS supports sessions with multiple trees.
  • Cross-Platform session file compatibility.

Version 2.0.1 (April 30, 2012)

  • Snapping charts to tree nodes
  • Restoring charts to default placement over locations
  • Border of pie charts can now be set to reflect the colour of their associated location marker
  • "Hidden" line style for chart dropline
  • Improved rendering of multifurcating trees

Version 2.0.0 (March 29, 2012)

  • Serialization (Save and Open sessions to/from GenGIS *.GGIS session files)
  • Python Plugins and Plugin Manager
  • Additional support for RPy2
  • Plots showing results of Monte Carlo permutation test
  • Comprehensive linear axes analysis for testing all possible linear axes
  • Show/Hide text under user interface buttons
  • Updated Legend Panels (Layers, Locations and Sequences)
  • Welcome Screen
  • Updated Help->About dialog
  • Updated documentation
